## 🛠 Features
- **Advanced CSS Grid:** Implements `grid-template-areas` to create a 3-column newspaper structure.
- **Glassmorphism UI:** Modern "frosted glass" effect using `backdrop-filter: blur()`.
- **Neon Aesthetics:** Custom glowing typography and interactive glitch effects on hover.
- **Responsive Media:** Optimized images using `object-fit: cover` to prevent distortion in grid cells.
- **Typography:** Hybrid font pairing (Orbitron for headers & Roboto Mono for data-style text).

## 📂 Project Structure
- `index.html` - Semantic HTML5 markup.
- `styles.css` - Custom CSS with Grid, Flexbox, and Animations.
- `assets/` - Images and icons (optional).

## 💡 What I Learned
During this project, I practiced:
1. Mapping complex layouts using names in `grid-template-areas`.
2. Handling image aspect ratios within fixed grid dimensions.
3. Creating a cohesive dark-mode UI with high-contrast neon accents.
